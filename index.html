Understanding Number Systems

Numbering systems use multiple digits to express any number through the use of a base. Each digit in a multi-digit number has a specific place value. This place value is determined by the base, raised to a power that corresponds to the digit's sequential position (starting from 0 on the right).

We are most accustomed to the Decimal System, where the base is 10 and we use ten numeral symbols (0-9). In this application, we explore three other critical systems:

  • Binary (Base 2): Uses symbols 0 and 1.
  • Octal (Base 8): Uses symbols 0 to 7.
  • Hexadecimal (Base 16): Uses symbols 0-9 and A-F.

These alternative systems—especially binary—are the foundation of digital electronics. They allow complex numerical data to be represented by the High/Low (or On/Off, 1/0) voltage values across a set of physical nodes in a computer's circuitry.

The Grouping Shortcut: Linking Binary to Octal and Hexadecimal

You can quickly convert between binary and the octal or hexadecimal systems without ever calculating the decimal value. This is done by grouping binary digits:

  • For Octal: Group binary digits into sets of 3. Because 23 = 8, a 3-bit binary number has exactly 8 possible states (000 to 111). This maps perfectly, one-to-one, with the 8 symbols of the octal system (0 to 7).
  • For Hexadecimal: Group binary digits into sets of 4. Because 24 = 16, a 4-bit binary number has exactly 16 possible states (0000 to 1111), mapping perfectly to the 16 symbols of hexadecimal (0 to F).
