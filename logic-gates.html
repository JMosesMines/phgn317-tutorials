Understanding Digital Logic Gates

Logic gates are the physical structural building blocks of integrated digital circuits. They execute fundamental Boolean algebra functions on one or more binary inputs to generate a singular binary output value.

Core Hardware Gate Profiles

Gate Schematic Symbol Boolean Expression Functional Rule
AND Y = A · B Output is 1 only if ALL inputs are 1.
OR Y = A + B Output is 1 if AT LEAST ONE input is 1.
NOT Y = A' Inverts the input value completely.

Derived Gates (NAND, NOR, XOR)

Placing an inversion bubble at the output node creates NAND and NOR gates, which flip the default results of AND and OR. The XOR (Exclusive OR) gate is unique: its output goes high if the inputs are different, but drops to low if both inputs match.
